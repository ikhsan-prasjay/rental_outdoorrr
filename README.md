**OutdoorGear Rental** adalah platform berbasis web yang memudahkan penyewaan peralatan camping dan pendakian gunung. Sistem ini mencakup fitur manajemen stok untuk admin dan alur pemesanan yang mudah bagi pelanggan.

## âœ¨ Fitur Utama

* **Autentikasi Pengguna**: Sistem Login dan Register yang aman.
* **Katalog Produk**: Tampilan daftar peralatan ("Gear") yang informatif.
* **Manajemen Produk (CRUD)**: Admin dapat menambah, mengedit, melihat detail, dan menghapus data produk.
* **Transaksi & Riwayat**: Pencatatan daftar transaksi penyewaan.
* **Sistem Pembayaran**: Antarmuka untuk memproses pembayaran sewa.
